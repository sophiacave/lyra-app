## LEARNING SCIENCE ENGINE — BAKED INTO EVERY FRAME

### Mayer's Multimedia Principles (Non-Negotiable)
| Principle | Production Rule |
|-----------|----------------|
| **Multimedia** | NEVER voice-only or visual-only. Every second has BOTH voice + matched visual. |
| **Coherence** | ZERO decorative elements. Every pixel teaches. If it doesn't serve comprehension, delete it. |
| **Signaling** | Visual cues (color change, motion, highlight) at EVERY key concept. The eye is guided, never wandering. |
| **Redundancy** | NEVER put on-screen text that duplicates spoken words exactly. Complement, don't repeat. |
| **Spatial Contiguity** | Labels TOUCH the thing they describe. No legends. No separate captions. |
| **Temporal Contiguity** | Show and tell SIMULTANEOUSLY. Never "as you can see in this diagram..." — show it AS you say it. |
| **Segmenting** | Complex ideas in 20-40 second chunks. Visual breath between segments (1-2s transition). |
| **Pre-training** | Define key terms BEFORE using them. Concrete example BEFORE abstract rule. |
| **Modality** | Narration (spoken) > on-screen text for explanations. Text only for labels, definitions, key terms. |
| **Personalization** | Conversational tone. "You" and "we." Never formal/academic register. |
| **Voice** | Human-sounding. Warm. Persona. Not robotic, not corporate. |
| **Image** | Voice quality matters more than seeing a face. A great voice > a talking head. |

### Cognitive Load Budget
Every scene has a cognitive load budget:
- **Intrinsic load**: The concept itself (we can't reduce this)
- **Extraneous load**: Bad design, clutter, confusion (ELIMINATE this)
- **Germane load**: Effort spent building mental models (MAXIMIZE this)

Production rules:
- Max 1 new concept per 20-40 seconds
- Max 3 visual elements on screen at once
- Progressive reveal: build diagrams piece by piece, never show complete
- Strategic silence: 1.5-2.5s pause after key insight for processing
- Pattern interrupt every 30-45s (change visual style, shift energy, ask question)

### Pacing Science
| Context | WPM | Pause After |
|---------|-----|-------------|
| Hook / opening | 155-165 | 0.5s |
| Teaching new concept | 135-145 | 1.5s after key point |
| Building complexity | 145-155 | 0.5s between steps |
| Revelation / aha moment | 120-130 | 2-3s silence |
| Emotional / awe moment | 110-125 | 2-3s + music swell |
| Closing / callback | 130-140 | 1.5s |

### Engagement Architecture
- **Curiosity gap** in first 5 seconds (open a question the brain NEEDS answered)
- **Concrete before abstract** ALWAYS (metaphor → mechanism → math, never reversed)
- **Prediction prompts** ("What do you think happens when...") before revealing answers
- **Callbacks** to opening hook in the final 15 seconds (pattern completion = satisfaction)
- **Emotional encoding**: Awe and surprise at key learning moments (amygdala activation = stronger memory formation)

## CINEMATIC CRAFT ENGINE — RED/ARRI GRADE

### The Cinema Look (What Separates YouTube from Film)

1. **Shallow depth of field** — subject sharp, background dreamy
2. **Motivated lighting** — every light source has a reason to exist
3. **Color science** — not Instagram filters, real film color response curves
4. **Film grain** — organic texture, not digital sterility
5. **Anamorphic characteristics** — horizontal flares, oval bokeh, edge breathing
6. **Letterboxing** — 2.39:1 widescreen for epic, 1.85:1 for intimate
7. **Motion blur** — 180-degree shutter angle (natural to the eye)
8. **Composition** — rule of thirds, leading lines, negative space, frame within frame
9. **Color palette discipline** ��� max 3-4 colors per scene, always motivated
10. **Sound design** — 50%+ of perceived "visual" quality is actually audio

### Cinematic Prompt Architecture (For Kling/AI Video)

Every B-roll prompt MUST specify all 7 elements:

```
[SUBJECT] + [CAMERA] + [LENS] + [LIGHTING] + [MOTION] + [COLOR/MOOD] + [NEGATIVE]
```

**SUBJECT**: Exactly what we see. Specific. Not abstract.
- ❌ "Neural pathways"
- ✅ "Single glowing neural fiber with translucent organic membrane"

**CAMERA**: Shot size + angle + height
- Extreme close-up, close-up, medium close-up, medium shot, medium wide, wide, extreme wide
- Eye level, low angle (heroic), high angle (vulnerable), bird's eye, Dutch angle

**LENS**: Focal length + characteristics
- "35mm anamorphic lens" / "85mm f/1.4" / "macro lens" / "wide-angle 16mm"
- "shallow depth of field" / "rack focus near to far" / "split diopter"

**LIGHTING**: Direction + quality + color
- "Single key light from camera left, warm tungsten, hard shadow"
- "Volumetric god rays through atmospheric haze"
- "Practical lighting from screens/monitors, cool blue spill"
- Cinematographer references: "Roger Deakins natural light" / "Bradford Young underexposed shadows" / "Emmanuel Lubezki golden hour"

**MOTION**: Camera movement + subject movement + speed
- Camera: "Slow dolly push-in" / "Steadicam tracking left" / "Crane rising" / "Static locked-off"
- Subject: "Pulse traveling left to right" / "Particles drifting upward" / "Structure assembling"
- Speed: "Slow motion 120fps feel" / "Real-time" / "Time-lapse"

**COLOR/MOOD**: Palette + atmosphere + film reference
- "Teal and orange, Blade Runner 2049 palette"
- "Desaturated cool blue with warm practicals, Sicario look"
- "High contrast, deep blacks, neon accent, Drive color palette"
- "Warm amber and earth tones, Terrence Malick golden hour"

**NEGATIVE**: What to exclude
- ALWAYS: "no faces, no text, no UI, no watermark, no stock photo look"
- Per-scene: "no human figures" / "no modern technology" / "no bright colors"

### FFmpeg Cinema Post-Processing Pipeline

Every video gets this treatment chain:

```
INPUT (AI-generated 1080p)
  ↓
1. LETTERBOX — Crop to 2.39:1 cinematic widescreen
   crop=1920:804:0:138
  ↓
2. COLOR GRADE — Apply cinema LUT
   lut3d=studio/luts/arri-logc-to-cinema.cube
  ↓
3. FILM GRAIN — 35mm organic texture overlay
   noise=c0s=8:c1s=6:allf=t+u
   (or composite real grain plate)
  ↓
4. VIGNETTE — Subtle darkening at edges
   vignette=PI/4
  ↓
5. SHARPEN — Gentle, cinema-appropriate
   unsharp=3:3:0.3
  ↓
6. ENCODE — H.264 high quality
   -c:v libx264 -crf 16 -preset slow
```

Full FFmpeg command:
```bash
ffmpeg -y -i input.mp4 \
  -vf "crop=1920:804:0:138,\
       lut3d='studio/luts/arri-cinema.cube',\
       noise=c0s=6:allf=t+u,\
       vignette=PI/5,\
       unsharp=3:3:0.3,\
       format=yuv420p" \
  -c:v libx264 -crf 16 -preset slow \
  -c:a copy \
  output_cinema.mp4
```

## THE VISUAL LAYER SYSTEM

Every scene is composed of stacked visual layers:

### Layer 1: BASE (Always present)
The primary visual. One of:
- **Cinematic B-roll** — Kling AI-generated, concept visualization
- **Motion graphic** — Remotion-rendered animated diagram/visualization
- **Manim scene** �� ManimCE mathematical animation
- **Color field** — Atmospheric gradient for voice-only moments

### Layer 2: OVERLAY (When teaching)
Added on top of base:
- **Kinetic typography** — Key terms appear as spoken, animate in, fade out
- **Animated diagram** �� Progressive reveal, building piece by piece
- **Data visualization** — Charts/graphs animating with the narration
- **Code blocks** — Syntax-highlighted, appearing line by line

### Layer 3: SIGNALING (Always active)
Visual cues that guide attention:
- **Highlight pulse** — Glow/brighten on the element being discussed
- **Arrow/pointer** — Appears at labels, connections
- **Color shift** — Element changes color when named
- **Scale change** — Element grows slightly when emphasized

### Layer 4: ATMOSPHERE (Constant)
The cinematic wrapper:
- **Letterbox bars** — 2.39:1 top/bottom
- **Film grain** — Subtle organic texture
- **Color grade** — LUT applied
- **Vignette** — Subtle edge darkening
- **Lower third** — Chapter title (first 3s of each segment)
