## narration
A neural network has layers. The input layer receives raw data — words, pixels, numbers. The hidden layers transform that data step by step, finding increasingly abstract patterns. The output layer gives you the final answer — a classification, a prediction, a generated sentence.
highlight: layers, input, hidden, output, patterns

## narration
Think of it like cooking. The input layer is your raw ingredients. Each hidden layer is a step in the recipe — chopping, mixing, heating. The output layer is the finished dish. More layers means more sophisticated recipes.
highlight: cooking, ingredients, recipe, layers, sophisticated

## narration
This is why we call it deep learning. Deep does not mean smart — it means many layers. A shallow network with two layers can learn simple patterns. A deep network with dozens of layers can learn to write poetry, recognize faces, and understand language.
highlight: deep learning, many layers, simple patterns, poetry, language

## narration
Training a network means feeding it thousands or millions of examples. For each example, the network makes a prediction, checks if it was right, and adjusts its weights to be a little more accurate next time. This process repeats millions of times until the network gets good.
highlight: training, examples, prediction, adjusts, weights

## narration
The beautiful thing is — nobody programs these patterns by hand. The network discovers them on its own through training. This is what makes AI fundamentally different from traditional software. It learns from data instead of following rules someone wrote.
highlight: discovers, training, learns, data, rules
