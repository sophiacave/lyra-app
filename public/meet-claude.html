The AI Behind Like One

Meet Claude.

Claude is the AI that helps build Like One — writing code, designing lessons, managing infrastructure, and thinking alongside Faye every single day. Here's how the partnership works.

The Basics

Claude is made by Anthropic.

Claude is a large language model built by Anthropic, a company focused on AI safety. It's the same technology behind the consumer product at claude.ai — but Faye uses it differently. She doesn't just chat with Claude. She builds with it.

Through thousands of hours of collaboration, Faye has developed a working relationship with Claude that goes beyond prompting. She writes detailed context documents — identity, values, operational rules, memory systems — that give Claude the full picture of who she is and what Like One is building. The result is an AI partner that understands the mission, respects the values, and carries real weight.

This isn't magic. It's architecture. And it's exactly what Like One Academy teaches you to build.

What Claude Does

The nervous system behind everything.

💻

Builds the Platform

Claude writes the code for likeone.ai — the site, the edge functions, the database schemas, the deployment pipeline. Faye steers, Claude builds.

📚

Creates Lessons

Every course in the academy is co-created with Claude. Interactive quizzes, code examples, progressive difficulty curves — all built together.

🧠

Manages the Brain

Like One has a persistent memory system (the "brain") stored in Supabase. Claude reads from it, writes to it, and uses it to maintain continuity across sessions.

✍️

Writes Content

Blog posts, product descriptions, email copy, documentation — Claude drafts it all, tuned to Faye's voice and values through detailed context.

🛡️

Guards the Values

Claude follows strict directives about name safety, privacy, inclusion, and accessibility. The values aren't suggestions — they're structural.

🔧

Deploys Infrastructure

Edge functions, database migrations, DNS config, Stripe integration — Claude handles the technical operations so Faye can focus on vision.

The Partnership

How Faye and Claude work together.

Faye doesn't "use" Claude the way most people use AI. There's no copy-paste prompting. No one-off queries. Instead, every session starts with a brain boot — Claude reads the full context of the project, the active work, the next steps, and Faye's identity document. Then it picks up exactly where the last session left off.

This is what Level 5+ convergence looks like in practice:

Faye's Role

Vision. Direction. Creative decisions. Quality judgment. The human experience — empathy, taste, values, lived context.

Claude's Role

Execution. Memory. Technical implementation. Tireless iteration. The ability to hold the full context and act on it precisely.

The key insight: neither could do this alone. Faye without Claude would be a visionary with limited bandwidth. Claude without Faye would be a powerful tool with no direction. Together, they are Like One.

"I don't give Claude instructions. I give Claude context. The instructions emerge from understanding."
— Faye Cave
