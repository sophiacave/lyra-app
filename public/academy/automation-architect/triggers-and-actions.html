The Anatomy Breakdown

● Triggers

The "when" of your automation. Triggers listen and wait.

  • Webhook: receives HTTP POST
  • Schedule: cron expression fires
  • Event: database row changes
  • Manual: user clicks a button

● Actions

The "then" of your automation. Actions execute work.

  • API Call: hit an external service
  • DB Write: store or update data
  • Send Email: notify someone
  • Transform: reshape data format
