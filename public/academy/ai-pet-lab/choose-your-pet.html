Design Your Agent

Every reinforcement learning agent is defined by its hyperparameters. Choose a configuration and observe how these values shape learning behavior.

Why Parameters Matter

A reinforcement learning agent's behavior is governed by a small set of critical numbers. The exploration rate ε determines how often the agent tries random actions versus exploiting what it already knows. A high epsilon means more exploration — the agent wanders broadly through the state space, potentially discovering surprising strategies but converging slowly. A low epsilon means the agent quickly settles on whatever looks best early, which risks missing superior long-term strategies.

The learning rate α controls how much new information overwrites old knowledge. An eager learner with high alpha rapidly adapts to recent rewards but can oscillate unstably. A cautious learner integrates experience gradually, building robust value estimates at the cost of slower adaptation.

Finally, the discount factor γ expresses how much the agent values future rewards relative to immediate ones. An agent with gamma near 1.0 is far-sighted, planning many steps ahead. An agent with low gamma is myopic, optimizing for the next reward alone. The interplay of these three parameters defines the agent's personality — its balance of caution and boldness, patience and greed.
