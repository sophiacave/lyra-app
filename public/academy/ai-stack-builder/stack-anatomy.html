Key Concept

Why Stack Thinking Matters

Most AI tutorials teach one tool in isolation. But real products need at least 5 tools working together. A chatbot needs a database to remember conversations, an API to call the LLM, a frontend to render messages, auth to know who's talking, and deployment to stay online. Stack thinking means choosing tools that play well together.

The Like One Stack

Throughout this course, we'll reference a production stack that's actually running in the wild:

// The Like One Production Stack
Database: Supabase // Postgres + Auth + Edge
Backend: Edge Functions (Deno) // Serverless, fast, cheap
AI: Claude via API + MCP // Agentic intelligence
Frontend: Next.js on Vercel // React, SSR, fast
Deploy: Vercel // git push = live
Automation: Make.com // Glue between everything
Payments: Stripe // Revenue from day 1
Email: Resend // Transactional + marketing
