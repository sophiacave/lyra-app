Key Concept: Signature Verification

Anyone can send a POST to your endpoint. Stripe signs every webhook with a secret. Always verify the signature.

import Stripe from "https://esm.sh/stripe@14"

const stripe = new Stripe(Deno.env.get("STRIPE_SECRET_KEY"))
const sig = req.headers.get("stripe-signature")
const body = await req.text()

const event = stripe.webhooks.constructEvent(
  body, sig, Deno.env.get("STRIPE_WEBHOOK_SECRET")
)

// If signature is invalid, constructEvent throws
// Now you can trust event.type and event.data
Pro Tips

Respond fast: Stripe expects a 200 response within 20 seconds. Do heavy processing asynchronously — acknowledge the webhook first, then process.

Idempotency: Stripe may send the same event multiple times. Always check if you've already processed an event ID before acting on it.

Retry logic: If your endpoint returns an error, Stripe retries with exponential backoff for up to 3 days. Design for this.
