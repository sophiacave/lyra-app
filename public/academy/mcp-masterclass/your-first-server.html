Key Pattern: server.tool()

Every MCP server follows the same pattern. The server.tool() method registers a tool with three arguments:

name A unique string identifier the AI uses to call this tool
schema A Zod schema defining the input parameters and their types
handler An async function that executes the tool logic and returns results
